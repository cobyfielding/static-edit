Welcome to static edit!

On this page, you can live-edit everything. Try clicking on me, or on the cats!

Don't forget to check the console for additional information about what is happening.

The setup is really easy:

<script src="static-edit.min.js"></script> <script> (function () { new StaticEdit.Editor({saveButton: true}) window.addEventListener('static_edit.edited', function (e) { // This is when the edition ends }) window.addEventListener('static_edit.editing', function (e) { // This is when the edition starts }) window.addEventListener('static_edit.saving', function (e) { // This is when the button "save" is pressed }) })() </script>
